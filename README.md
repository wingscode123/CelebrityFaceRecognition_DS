# CelebrityFaceRecognition_DS

Proyek ini membangun sistem klasifikasi gambar untuk mengenali tokoh selebriti terkenal yang sudah terlabeli dengan pendekatan data sains. roses meliputi scraping dataset gambar dari Google, pembersihan data dengan Numpy, OpenCV, dan Matplotlib, serta pengembangan model machine learning menggunakan Scikit-learn. odel dilatih dan diintegrasikan ke dalam sistem berbasis Python Flask dengan antarmuka web interatif
